Slower Than Sound

a keyboard-only FTL-inspired game about managing multiple systems aboard an attack submarine.

=== Controls ===
Shift+Q..........Overview screen
Shift+W..........Power distribution screen
Shift+E..........AI screen
Shift+R..........Fire control screen
Shift+A..........Engine screen
Shift+S..........Oxygen screen
Shift+D..........Bulkhead screen
Shift+F..........Weapons screen
Shift+T/Shift+G..LIDAR screen
