Slower Than Sound

a keyboard-only FTL-inspired game about managing multiple systems aboard an attack submarine.

=== Controls ===
Shift+Q..........Overview screen
Shift+W..........Power distribution screen
Shift+E..........AI screen
Shift+R..........Fire control screen

== Engine Controls ==
Shift+U...Set speed
Shift+J...Emergency stop/acclerate
Shift+I...Set heading
Shift+K...Emergency turn
Shift+O...Set depth
Shift+L...Emergency dive/surface

== Power/Oxy Controls ==
QWERASDF...Select System
U..........Increase electricity
J..........Decrease electricity
I..........Increase lubricant
K..........Decrease lubricant
O..........Increase coolant
L..........Decrease coolant

== Targeting and Fire Controls ==
U......Set LIDAR
I......Set enemy select
O......Lock enemy into torp
J......Select torpedo tube
K......Select torpedo type
L......Load/launch torpedo
